# LOKACIA.KZ — Описание проекта для Astana Hub

---

## 1. О проекте

**LOKACIA.KZ** — маркетплейс аренды локаций и спортивных площадок для съёмок, мероприятий, встреч и спорта в Казахстане. Платформа соединяет владельцев площадок (фотостудии, банкетные залы, лофты, этно-юрты, рестораны, офисы, футбольные поля, теннисные и падел-корты) с арендаторами — фотографами, продакшн-студиями, ивент-менеджерами, спортсменами и частными лицами.

Мы строим казахстанский аналог Peerspace (США, $100M revenue, 40 000 площадок) и Giggster (США, $1M страховка, фокус на production) — с адаптацией под локальный рынок: тенговые платежи, верификация по ИИН/БИН, интеграция с Kaspi Pay и Halyk Bank.

**Сайт:** https://lokacia.kz
**Юрлицо:** ТОО «Lokacia» (учредитель — Arvo Group Ltd, AIFC)
**Статус:** MVP задеплоен и работает. Мобильное приложение (React Native) в разработке — запуск в течение месяца.

## 2. Проблема

Рынок аренды площадок в Казахстане фрагментирован:

- **Для арендаторов:** поиск площадки занимает часы — звонки по Instagram, переписка в WhatsApp, нет стандартизированных описаний, нет гарантий качества. Невозможно сравнить цены и проверить отзывы в одном месте.

- **Для владельцев площадок:** нет единого канала привлечения клиентов. Продвижение через Instagram требует постоянных затрат на контент. Нет инструментов управления бронированиями, нет защиты от ущерба.

- **Для спортсменов:** бронирование футбольных полей и кортов — через звонки, без онлайн-расписания. Нет возможности сравнить цены и доступность в одном месте.

- **Доверие:** ни одна существующая платформа в регионе не предоставляет верификацию сторон, программу защиты или систему разрешения споров.

## 3. Решение

LOKACIA.KZ — полнофункциональная платформа с замкнутым циклом:

**Поиск → Верификация → Бронирование → Оплата → Защита**

### Для арендаторов:
- Каталог локаций и спортивных площадок с фильтрами, картой и мгновенным поиском
- Детальные карточки: фото-галерея, площадь, вместимость, оборудование, отзывы
- Онлайн-бронирование с выбором даты и расчётом стоимости
- Чат с владельцем (с фильтрацией контактов до подтверждения бронирования)
- Программа защиты при возникновении проблем

### Для владельцев (хостов):
- Бесплатное размещение листинга с неограниченным количеством фото
- Панель управления: бронирования, доходы, календарь, скидки
- ID-верификация (бейдж доверия повышает конверсию)
- Защита имущества через залоговый депозит
- Модерация и поддержка платформы

### Trust & Safety (ключевое отличие):
- OTP-верификация телефона (flash-call через smsc.kz)
- ID-верификация: физлицо (ИИН + удостоверение) и юрлицо (БИН + свидетельство)
- Модерация листингов (pending → approve/reject)
- Программа защиты с залоговым депозитом и порталом претензий
- Фильтрация контактных данных в чате (анти-обход платформы)

## 4. Технологии

| Компонент | Технология |
|-----------|-----------|
| Frontend | Next.js 16, React 19, TypeScript, Tailwind CSS 4 |
| Backend / БД | Supabase (PostgreSQL + Row Level Security + Realtime) |
| Инфраструктура | Hetzner Cloud, Docker, CI/CD (GitHub Actions) |
| Безопасность | Zod validation, Rate limiting, RLS, OTP-верификация |
| Платежи | Kaspi Pay, Halyk Bank |
| Коммуникации | Resend (email), smsc.kz (flash-call OTP) |
| Тестирование | Vitest, Testing Library — 47 тестов, TDD |
| Мобильный | PWA (Service Worker, Web App Manifest) + React Native / Expo |

**Архитектура:** монолитный Next.js с серверными и клиентскими компонентами, Supabase как BaaS (аутентификация, база данных, хранилище файлов, realtime). Дизайн-система задокументирована в DESIGN.md с переиспользуемыми UI-компонентами.

**Мобильное приложение:** React Native / Expo — в активной разработке, запуск в течение месяца.

## 5. Бизнес-модель

**Двусторонняя комиссия** с каждой транзакции — модель Peerspace/Giggster:

| Параметр | Значение |
|----------|----------|
| Комиссия с хоста | 15% от суммы бронирования |
| Комиссия с арендатора | 7.5% (сервисный сбор) |
| Промо-период | 0% комиссия при запуске |
| Размещение | Бесплатно для всех хостов |

**Эффективная комиссия платформы: 22.5% от каждого бронирования.**

**Дополнительные источники дохода (roadmap):**
- Promoted listings (платное продвижение в каталоге)
- Featured-бейджи для хостов
- B2B-подписка для ивент-агентств и спортивных клубов

## 6. Рынок

### Глобальный контекст

Мировой рынок аренды площадок для мероприятий — $100B+. Лидеры:

| Платформа | Страна | Revenue | Площадок | Модель |
|-----------|--------|---------|----------|--------|
| Peerspace | США | $100M | 40 000 | 15% host + 5-15% renter |
| Giggster | США | — | 20 000+ | 15% host + 5-15% renter |
| Bash Today | Россия | — | 3 000 | Каталог без оплаты |
| Squares.kz | Казахстан | — | 2 000 | Бесплатный каталог + Boost |

### Казахстан

- Рынок ивент-услуг растёт на 15-20% ежегодно
- Алматы: 500+ площадок (фотостудии, банкетные залы, лофты) + 300+ спортивных объектов
- Астана: 300+ площадок + 200+ спортивных объектов
- Шымкент: 150+ площадок + 100+ спортивных объектов
- Бум падел-кортов и мини-футбольных арен (2024-2026)
- Production-сегмент растёт (кино, реклама, контент)

### Конкурентная среда в Казахстане

**Squares.kz** — единственный прямой конкурент. 2000 локаций, 20 городов. Однако:
- Нет онлайн-оплаты через платформу
- Нет верификации сторон (ИИН/БИН)
- Нет программы защиты и залогов
- Нет системы споров и претензий
- Нет спортивного сегмента
- Модель каталога, а не маркетплейса с полным циклом

**LOKACIA.KZ** — единственная платформа в регионе с trust & safety, полным транзакционным циклом и охватом event + спортивного сегмента.
