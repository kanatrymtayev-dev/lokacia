# LOKACIA.KZ — Описание проекта для Astana Hub

---

## 1. О проекте

**LOKACIA.KZ** — маркетплейс аренды локаций для съёмок, мероприятий и встреч в Казахстане. Платформа соединяет владельцев площадок (фотостудии, банкетные залы, лофты, этно-юрты, рестораны, офисы) с арендаторами — фотографами, продакшн-студиями, ивент-менеджерами и частными лицами.

Мы строим казахстанский аналог Peerspace (США, $100M revenue, 40 000 площадок) и Giggster (США, $1M страховка, фокус на production) — с адаптацией под локальный рынок: тенговые платежи, верификация по ИИН/БИН, интеграция с Kaspi Pay.

**Сайт:** https://lokacia.kz
**Статус:** MVP задеплоен и работает. Подключение онлайн-оплаты — в процессе.

---

## 2. Проблема

Рынок аренды площадок в Казахстане фрагментирован:

- **Для арендаторов:** поиск площадки занимает часы — звонки по Instagram, переписка в WhatsApp, нет стандартизированных описаний, нет гарантий качества. Невозможно сравнить цены и проверить отзывы в одном месте.

- **Для владельцев площадок:** нет единого канала привлечения клиентов. Продвижение через Instagram требует постоянных затрат на контент. Нет инструментов управления бронированиями, нет защиты от ущерба.

- **Доверие:** ни одна существующая платформа в регионе не предоставляет верификацию сторон, программу защиты или систему разрешения споров.

---

## 3. Решение

LOKACIA.KZ — полнофункциональная платформа с замкнутым циклом:

**Поиск → Верификация → Бронирование → Оплата → Защита**

### Для арендаторов:
- Каталог локаций с фильтрами, картой и мгновенным поиском
- Детальные карточки: фото-галерея, площадь, вместимость, оборудование, отзывы
- Онлайн-бронирование с выбором даты и расчётом стоимости
- Чат с владельцем (с фильтрацией контактов до подтверждения бронирования)
- Программа защиты при возникновении проблем

### Для владельцев (хостов):
- Бесплатное размещение листинга с неограниченным количеством фото
- Панель управления: бронирования, доходы, календарь, скидки
- ID-верификация (бейдж доверия повышает конверсию)
- Защита имущества через залоговый депозит
- Модерация и поддержка платформы

### Trust & Safety (ключевое отличие):
- OTP-верификация телефона (flash-call через smsc.kz)
- ID-верификация: физлицо (ИИН + удостоверение) и юрлицо (БИН + свидетельство)
- Модерация листингов (pending → approve/reject)
- Программа защиты с залоговым депозитом и порталом претензий
- Фильтрация контактных данных в чате (анти-обход платформы)

## 4. Технологии

| Компонент | Технология |
|-----------|-----------|
| Frontend | Next.js 16, React 19, TypeScript, Tailwind CSS |
| Backend / БД | Supabase (PostgreSQL), Row Level Security |
| Хостинг | Hetzner Cloud (Германия), Docker |
| CI/CD | GitHub Actions → SSH → Docker Compose |
| Платежи | PayBox.money (в подключении), Kaspi Pay |
| SMS/OTP | smsc.kz (flash-call верификация) |
| Email | Resend (транзакционные письма) |
| Тестирование | Vitest, Testing Library, TDD (32+ теста) |
| PWA | Service Worker, Web App Manifest |

**Архитектура:** монолитный Next.js с серверными и клиентскими компонентами, Supabase как BaaS (аутентификация, база данных, хранилище файлов, realtime). Дизайн-система задокументирована в DESIGN.md с переиспользуемыми UI-компонентами.

**Мобильное приложение:** в разработке (React Native / Expo). Текущая PWA-версия позволяет установить приложение на главный экран телефона и работать офлайн.

## 6. Рынок

### Глобальный контекст

Мировой рынок аренды площадок для мероприятий — $100B+. Лидеры:

| Платформа | Страна | Revenue | Площадок | Модель |
|-----------|--------|---------|----------|--------|
| Peerspace | США | $100M | 40 000 | 15% host + 5-15% renter |
| Giggster | США | — | 20 000+ | 15% host + 5-15% renter |
| Bash Today | Россия | — | 3 000 | Каталог без оплаты |
| Squares.kz | Казахстан | — | 2 000 | Бесплатный каталог + Boost |

### Казахстан

- Рынок ивент-услуг растёт на 15-20% ежегодно
- Алматы: 500+ площадок (фотостудии, банкетные залы, лофты)
- Астана: 300+ площадок
- Production-сегмент растёт (кино, реклама, контент)

### Конкурентная среда в Казахстане

**Squares.kz** — единственный прямой конкурент. 2000 локаций, 20 городов. Однако:
- Нет онлайн-оплаты через платформу
- Нет верификации сторон (ИИН/БИН)
- Нет программы защиты и залогов
- Нет системы споров и претензий
- Модель каталога, а не маркетплейса с полным циклом

**LOKACIA.KZ** — единственная платформа в регионе с trust & safety и полным транзакционным циклом.

- Нулевая конкуренция в нише
- Русский язык — минимальные затраты на локализацию
- Production-сегмент растёт (кино, реклама)

**Техническая готовность к масштабированию:**
- Мультиязычность уже встроена (RU/EN/KZ)
- Архитектура позволяет добавить новые города и валюты
- Supabase масштабируется горизонтально
