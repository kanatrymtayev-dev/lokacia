# СЛАЙД 1 — ПРОБЛЕМА

## Как сегодня арендуют площадки в Казахстане

- Поиск через Instagram и WhatsApp — часы переписки
- Нет единого каталога с ценами и отзывами
- Нет онлайн-оплаты и бронирования
- Нет верификации — непонятно кому доверять
- Нет защиты при ущербе или отмене
- Спортивные площадки: звонки, нет онлайн-расписания

**Рынок аренды площадок (event + спорт): ~65 млрд тенге/год в КЗ**

# СЛАЙД 2 — РЕШЕНИЕ

## LOKACIA.KZ — полный цикл аренды

**Поиск → Верификация → Бронирование → Оплата → Защита**

Единая платформа, где:
- Арендатор находит площадку за 5 минут
- Хост получает клиентов без затрат на маркетинг
- Обе стороны защищены: верификация, залог, арбитраж

**Два сегмента в одной платформе:**
- Event & Production: студии, залы, лофты, юрты
- Спорт: футбольные поля, теннисные и падел-корты, бассейны

**Сайт:** lokacia.kz (MVP работает)
