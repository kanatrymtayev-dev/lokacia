## Фотостудии (15 штук — приоритет #1)

Фотостудии — идеальные первые хосты: они уже сдают в аренду, понимают модель, нужен только новый канал продаж.

| # | Где искать | Как найти |
|---|-----------|-----------|
| 1-5 | Instagram: поиск "фотостудия Алматы" | Топ-5 по подписчикам |
| 6-10 | 2GIS: "фотостудия" в Алматы | Первые 5 с отзывами |
| 11-15 | Google: "аренда фотостудии Алматы" | Первая страница выдачи |

**Районы концентрации:** Алмалинский район (центр), Медеуский район, Бостандыкский район

**Скрипт для фотостудий:**
"Здравствуйте! Я из LOKACIA.KZ — это новая платформа для аренды площадок, типа Airbnb для студий. Вы сдаёте студию через Instagram? А если бы вам приходили дополнительные клиенты с сайта — бесплатно, комиссия только когда есть реальная бронь? Можно показать как это выглядит?"

## Трекер обхода (Google Sheets)

Создать таблицу с колонками:

| Дата | Название | Тип | Адрес | Контакт | Статус | Следующий шаг |
|------|---------|-----|-------|---------|--------|--------------|
| | | | | | Визит / Созвон / Отказ / Зарегистрирован | |

**Статусы:**
- **Новый** — ещё не контактировали
- **Визит запланирован** — договорились о встрече
- **Визит проведён** — были на месте, показали платформу
- **Заинтересован** — хост хочет, но нужно время/помощь
- **Зарегистрирован** — создал аккаунт на lokacia.kz
- **Листинг создан** — площадка опубликована (или на модерации)
- **Отказ** — не заинтересован (причина?)

## KPI для Sales Manager

**Неделя 1-2 (онбординг):**
- Изучить платформу, создать тестовый листинг
- Составить список 50 площадок с адресами и контактами
- 10 первых визитов

**Месяц 1:**
- 40+ визитов
- 10+ зарегистрированных хостов
- 5+ опубликованных листингов

**Месяц 2:**
- 15+ новых хостов
- 10+ новых листингов
- Первые переговоры со страховой / Kaspi

**Месяц 3:**
- 20+ новых хостов
- Первые бронирования через платформу
- Партнёрство со страховой в процессе
